# Contributing — monspell fork of lang-khk

This is a **close fork** maintained for [monspell](https://github.com/abadrangui/monspell), following the same discipline as monspell-win (monspell ADR-0002):

- **Upstream-first.** Any generally useful change is PR'd upstream before (or instead of) landing here. Fork-local commits are limited to monspell-specific needs.
- **Rebase on upstream every 2–3 months.** Divergence budget is deliberately small.
- **Every deliberate divergence is logged** in `docs/fork-divergence.md` with a revisit trigger.

Upstream: https://github.com/giellalt/lang-khk
